Capability Map

从图像、序列、结构到候选分子的一张能力地图

这些 skill 不是孤立工具，而是可组合的研究链路：先识别和建模，再设计和筛选，随后用性质、合成与可开发性指标做决策。

成药判断ADMET、SAScore、逆合成、BioPhi 和 SAP 风险检查。

Small Molecule Track

适合先导发现、先导优化、商业库筛选、文献分子提取以及候选分子的早期成药性评估。

Structure Intelligence

结构预测与结合位点预测，给设计任务确定坐标系

结构类 skill 的价值在于把自然语言、序列、PDB ID 或上传结构转成可执行的模型、口袋、残基概率和 docking box。

structure-prediction

支持蛋白、核酸、配体与复合物预测；可在需要时带上 MSA、template 或 restraint 输入，用于后续 docking、筛选和结构设计。

Protein-ligandProtein-proteinProtein-DNA/RNA

Antibody Engineering

抗体工程：编号、人源化、建模、突变扫描与精准重设计

antibody-engineering skill 把 ANARCI、BioPhi、IgFold、FoldX 和 Rosetta 串成端到端候选优化闭环。

1

序列编号

ANARCI 用 IMGT、Kabat 等体系解析 VH/VL 区域与 CDR 边界。

2

人源化

BioPhi 评估 humanness 与 OASis 风险，生成 Sapiens 或 CDR grafting 变体。

3

结构建模

IgFold 预测亲本与候选结构，Rosetta FastRelax 进行局部放松。

4

能量扫描

FoldX 对稳定性、AlaScan、PositionScan 和 AnalyseComplex 做高通量初筛。

5

精准设计

Rosetta FastDesign 与 InterfaceAnalyzer 复核界面质量，SAP Score 控制聚集风险。

De-risking

先降风险再优化

先明确 CDR/FR 边界和免疫原性风险，再决定哪些位置允许突变。

Energy

亲和力与稳定性并行

候选筛选同时关注结合能和折叠稳定性，避免单目标优化带来的开发风险。

Panel

输出 Top 10-20 候选

最终候选综合 FoldX、Rosetta、SAP、IgFold 和 BioPhi 指标，形成可推进 panel。

Composable Workflow

推荐组合：从资料输入到候选决策的一条可复用路径

每个 skill 都通过 SciMiner 内部 API 调用，返回统一任务状态和 share_url。把结果链接沉淀下来，就能形成可追踪、可复审的研究记录。

1

提取输入

从图片提取 SMILES，或从 UniProt、序列、PDB、上传文件建立起始材料。

2

建立结构

结构预测补齐未知复合物，结合位点预测给出口袋与残基概率。

3

生成候选

按任务选择小分子、肽、蛋白、抗体或纳米抗体设计工具。

4

筛选评分

虚拟筛选、Gnina、FoldX、Rosetta、AfCycDesign 等工具帮助候选排序。

5

成药收敛

ADMET/PKPD、SAScore、逆合成和可开发性指标决定下一轮实验优先级。

结构驱动路线

PDB/结构文件已知

binding-site-prediction → Get Box → small-molecule-design 或 virtual-screening → admet-pkpd → synthesis-evaluation。

序列驱动路线

只有靶点名称或序列

Get Protein Sequence / structure-prediction → binding-site-prediction → protein-design / peptide-design → 结构与性质验证。

统一安装、统一鉴权、统一结果链接

从 SciMiner 获取免费 API key 后，设置 SCIMINER_API_KEY。调用时以 X-Auth-Token header 发送。
