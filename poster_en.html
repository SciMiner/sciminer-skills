Antibody Engineering

Antibody Engineering: Numbering, Humanization, Modeling, Mutation Scanning, and Precision Redesign

The antibody-engineering skill connects ANARCI, BioPhi, IgFold, FoldX, and Rosetta into an end-to-end candidate optimization loop.

1

Sequence Numbering

ANARCI parses VH/VL regions and CDR boundaries using IMGT, Kabat, and related schemes.

2

Humanization

BioPhi evaluates humanness and OASis risk, then generates Sapiens or CDR-grafting variants.

3

Structure Modeling

IgFold predicts parental and candidate structures, while Rosetta FastRelax performs local relaxation.

4

Energy Scanning

FoldX runs high-throughput first-pass screening for stability, AlaScan, PositionScan, and AnalyseComplex.

5

Precision Design

Rosetta FastDesign and InterfaceAnalyzer rescore interface quality, while SAP Score controls aggregation risk.

De-risking

De-Risk Before Optimizing

Clarify CDR/FR boundaries and immunogenicity risk before deciding which positions can mutate.

Energy

Affinity and Stability in Parallel

Screen candidates for binding energy and folding stability together to avoid single-objective development risk.

Panel

Output Top 10-20 Candidates

Final candidates balance FoldX, Rosetta, SAP, IgFold, and BioPhi metrics into a panel ready to advance.

Composable Workflow

Recommended Composition: A Reusable Path from Inputs to Candidate Decisions

Each skill calls the SciMiner internal API and returns unified task status plus a share_url. Preserving result links creates traceable, reviewable research records.

1

Extract Inputs

Extract SMILES from images, or build starting material from UniProt, sequences, PDB IDs, and uploaded files.

2

Build Structure

Structure prediction fills unknown complexes; binding-site prediction provides pockets and residue probabilities.

3

Generate Candidates

Choose small-molecule, peptide, protein, antibody, or nanobody design tools based on the task.

4

Screen and Score

Virtual screening, Gnina, FoldX, Rosetta, AfCycDesign, and related tools help rank candidates.

5

Drugability Convergence

ADMET/PKPD, SAScore, retrosynthesis, and developability indicators determine the next experimental priorities.

Structure-Driven Route

PDB or Structure File Available

binding-site-prediction -> Get Box -> small-molecule-design or virtual-screening -> admet-pkpd -> synthesis-evaluation.

Sequence-Driven Route

Only Target Name or Sequence Available

Get Protein Sequence / structure-prediction -> binding-site-prediction -> protein-design / peptide-design -> structure and property validation.

Install & Invoke

Unified Installation, Authentication, and Result Links

OpenClaw can install from the skill marketplace or add this repository. Claude Code can install sciminer-skills on request. Every skill requires a SciMiner API key.

Credential

Environment Variable

Get a free SciMiner API key, then set SCIMINER_API_KEY. Calls send it as the X-Auth-Token header.

https://sciminer.tech/utilitySCIMINER_API_KEY
Result

Task Status and Share Links

The internal API returns a task_id. Poll the result endpoint to get SUCCESS/FAILURE, result content, and an online share_url.

task_idstatusshare_url
BASE_URL = "https://sciminer.tech/console/api"
headers = {"X-Auth-Token": SCIMINER_API_KEY, "Content-Type": "application/json"}

payload = {
    "provider_name": "PocketXMol",
    "tool_name": "sbdd_gpu_sbdd_gpu_post",
    "parameters": {"protein": "<FILE_ID>", "binding_site": "Center:10,12,8;Size:20"}
}

POST /v1/internal/tools/invoke  →  task_id
GET  /v1/internal/tools/result  →  status, result, share_url
