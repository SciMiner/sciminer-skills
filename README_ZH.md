SciMiner 技能集

一套基于 SciMiner API 构建的精简技能集合。

安装
- OpenClaw：在 OpenClaw 技能商店中安装或将本仓库添加为技能源。
- Claude Code：在 Claude Code 对话中请求安装 "sciminer-skills"，或通过以下 ClawHub 页面直接安装：
	- https://clawhub.ai/sciminer/admet-pkpd
	- https://clawhub.ai/xiongzhp/structure-prediction
	- https://clawhub.ai/xiongzhp/peptide-design
	- https://clawhub.ai/xiongzhp/protein-design

使用
- 每个技能子文件夹包含 SKILL.md，详见 admet-pkpd、peptide-design、protein-design、structure-prediction。


备注
- 这些技能依赖 SciMiner API，可能需要 API 凭证。
